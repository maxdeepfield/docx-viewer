# Code of Conduct

We follow the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) to ensure a welcoming and inclusive community.

- Be respectful and constructive.
- Assume positive intent; clarify before escalating.
- Harassment, discrimination, or exclusionary behavior is not tolerated.

Issues or incidents? Email the maintainers or open a private issue so we can address it promptly.
